# Shopping-List-UI

> This project was done as coursework so it cannot be openly shared as per Policy 71 of the University of Waterloo, the source code is available upon request.

A responsive shopping list interface built with **Preact**, **TypeScript**, and **Vite**. The project focuses on item entry, category-based organization, checklist interactions, category editing, and expense summaries in a compact web UI.

## Demo

https://github.com/user-attachments/assets/38967b5c-edb1-4db4-8425-66c86f116cf8

## Features

- Add grocery items with quantity and category selection
- Automatically group items by category
- Update quantities directly from the list
- Mark items as bought with visual completion feedback
- Edit item categories through a modal view
- View bought-item expense summaries and total cost
- Use a compact responsive layout for a focused single-page experience

## Tech Stack

- Preact
- TypeScript
- Vite
- Preact Signals
- HTML/CSS
